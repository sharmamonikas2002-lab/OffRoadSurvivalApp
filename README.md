#  OffRoadSurvivalApp

React Native mobile app prototype for an extreme off-road adventure platform (inspired by real adventure motorsport events in India).

Built with **React Native + Expo** as a personal project to practice mobile UI/UX and showcase full-stack capabilities.

## Features
- Dramatic hero section with high-impact background and tagline
- Featured upcoming event card (The Survival Run 2026)
- Modern 2-column grid layout displaying intense off-road action gallery
- Fully responsive dark theme optimized for mobile
- Smooth scrolling and touch interactions
- Real adventure imagery (mud, mountains, water crossings, desert)

## Tech Stack
- React Native (Expo)
- JavaScript / TypeScript-ready
- Native components: FlatList, ImageBackground, TouchableOpacity
